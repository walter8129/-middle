Source Code Management and Easy Remote Communication with Middle

What is Middle?

Middle is an easy library which caches modules in the server, allowing for security and replicates shared modules as well as built in communication methods for quick remotes.

local Middle = require(...)

LOAD A STORED PACKAGE
Middle:LoadPackage(package-name)
LOAD A COMMON FILE
Middle:LoadCommon(package-name)

CREATE OR GET A REMOTE
Middle:GetRemoteEvent(event-name)
CREATE OR GET A REMOTE FUNCTION
Middle:GetRemoteFunction(function-name)
